# RS Node.js CRUD API

## Installation

- npm install

## Available Scripts

- `npm run start:dev` — Run the app in development mode using nodemon. Automatically restarts on file changes.

- `npm run build` — Compile TypeScript source code to JavaScript in the dist/ directory.

- `npm run start:prod` — Build the project and run the production-ready server from dist/server.js.

## Project Structure

 - `src/` — Source code
 - `dist/` — Compiled JavaScript output (after build)
 - `package.json`
 - `tsconfig.json`
